Tecnologias que utilizo:

- React:
  Framework javascript desenvolvida pelo facebook amplamente utilizada para desenvolvimento front-end.

- Node.js:
  O Node.js pode ser definido como um ambiente de execução Javascript server-side.

- TypeScript:
  É uma linguagem criada pela Microsoft e nada mais é que um superset do ECMAScript 6 que, por sua vez, é um superset do - ECMAScript 5, que usamos mais frequentemente como base para nosso clássico JavaScript.

- Python voltado a analise de dados:
  A análise de dados é um conjunto de técnicas que permite extrair informação sobre um conjunto de dados.

- Linguagem R:
  A linguagem R é largamente usada entre estatísticos e analistas de dados para desenvolver software de estatística e análise de dados.

- PostegresSQL:
  Database Server é um sistema de banco de dados poderoso e em código aberto para Windows, Mac e Linux. Ele possui mais de 15 anos de desenvolvimento ativo e uma arquitetura que ganhou uma forte reputação devido a sua estabilidade e integridade de dados.
- MySQL:
  é um sistema gerenciador de banco de dados relacional de código aberto usado na maioria das aplicações gratuitas para gerir suas bases de dados.

- Git:
  é um sistema de controle de versão de arquivos. Através deles podemos desenvolver projetos na qual diversas pessoas podem contribuir simultaneamente no mesmo, editando e criando novos arquivos e permitindo que os mesmos possam existir sem o risco de suas alterações serem sobrescritas.

Ja trablhei com a metodologia scrum com alguns projetos, tenho experiencia em trabalhar em equipes.
